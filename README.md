> **One blog post. Five channels. Zero extra work.**

Content teams spend hours manually rewriting an article into Twitter threads, LinkedIn carousels, and newsletters. Each channel gets a rushed, inconsistent version. The **Content Repurposing Agent** solves this by scraping the source URL deterministically and atomizing it into MECE, channel-native content that actually performs.

---

### 💎 Enterprise Features

* **🌐 Live Article Scraping:** Uses `scrape_article.py` to fetch raw text from any public URL — no hallucinated summaries.
* **🐦 X/Twitter Thread:** Outputs a 5-part MECE thread where each tweet builds on the last.
* **📸 LinkedIn Carousel:** Generates slide-by-slide text breakdowns optimized for professional engagement.
* **📧 Newsletter Hook:** A 150-word synthesis with a punchy opener, core insight, and CTA.
* **🔗 Brand Voice Locking:** All outputs are adapted to the brand tone you specify upfront.
